# Spigcord

A Discord Bot wrapper similar to Bukkit or Spigot.

## Installation

For the package use:

```shell
npm i spigcord
```

**Coming Soon**

For the CLI project generator:

```shell
npm i spigcord-cli
```

## Usage

```ts
import {
 Bot
} from "spigcord";

class MyNewBot extends Bot {
 public onEnable(): void {
  console.log("The bot is online!!");
 }
}

new MyNewBot();
```
